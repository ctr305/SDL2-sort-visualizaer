# SDL2-sort-visualizer
Visualize sorting arlgorithms in SDL2
# Usage
Compilation:
`make`

Deleting the executable:
`make clean`

Launching the visualizer:
`./sort.out ARGS`

Arguments:

|Argument|Visualized Algorithm|
|---|---|
|simple|Simple sorting algorithm|
|quick|QuickSort|
|merge|MergeSort|
|bubble|BubbleSort|
